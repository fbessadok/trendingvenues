Welcome to 'Trending Venues'

Trending Venues is a tool that helps you discover places around you that trends right now in social media communities.

Trending Venues is built with these technologies:

  • Play Framework 2, Scala language
  • Foursquare API
  • Ink (sapo) UI framework
  • Less CSS
  • Google Maps API v3
  • jQuery 1.9.0

Credits for the (free to use anywhere) earth image in favicon goes to brightmix.

Trending Venues is built by Firas Bessadok. Feel free to fork the project on github, report issues, and make pull requests.
